## @first

* @first()

随机生成一个常见的英文名。

```js
@first()
// => "Nancy"
```

## @last

* @last()

随机生成一个常见的英文姓。

```js
@last()
// => "Martinez"
```

## @name

* @name()
* @name( middle )

随机生成一个常见的英文姓名。

#### middle <span class="no-needed">`可选`</span>

布尔值。指示是否生成中间名。

```js
@name()
// => "Larry Wilson"
@name(true)
// => "Helen Carol Martinez"
```
